## Tugas per Anggota (Ringkasan)

### Anggota 1: Frontend & UI/UX

**Tanggung Jawab:**
- Pengembangan antarmuka pengguna
- Implementasi komponen shadcn/ui
- Styling dengan TailwindCSS
- Animasi dengan Framer Motion
- Responsif design

**Task List:**
1. Setup project Next.js + TailwindCSS + shadcn/ui
2. Buat halaman awal (Home Page)
3. Buat halaman pemilihan keputusan (Decision Page - 7 kategori)
4. Buat halaman simulasi (Simulation Page)
5. Buat halaman hasil masa depan (Result Page)
6. Buat halaman perbandingan (Comparison Page)
7. Implementasi navigasi antar halaman & menu
8. Testing responsive design
9. UI polish & animations

### Anggota 2: Logic & Integrasi AI

**Tanggung Jawab:**
- Decision Engine logic
- State management
- Integrasi PixVerse API
- Mapping scenario ke video
- Data management

**Task List:**
1. Rancang struktur state untuk user decisions
2. Implementasi Decision Engine
3. Rancang mapping scenario ke video & mock data
4. Setup integrasi PixVerse API
5. Implementasi logic multiple outcome simulation
6. Implementasi perbandingan multiple future
7. Local storage untuk menyimpan hasil simulasi
8. Testing semua logic flow

### Anggota 3: Video Production & Storytelling

**Tanggung Jawab:**
- Storyboard creation
- Shot planning
- Video generation via PixVerse
- Prompt engineering
- Video editing & finalization

**Task List:**
1. Brainstorming storyboard untuk semua skenario
2. Buat shot list untuk setiap video
3. Generate video dengan PixVerse (min 30 detik)
4. Evaluasi & iterasi video
5. Finalisasi semua video
6. Dokumentasi prompt yang digunakan
7. Integrasi video ke aplikasi

## Definition of Done (DoD)

Setiap task dianggap selesai jika:
- Code di-review oleh minimal 1 anggota tim lain
- Semua unit test lulus
- Tidak ada error pada build
- Fitur berjalan sesuai spesifikasi
- Dokumentasi di-update
