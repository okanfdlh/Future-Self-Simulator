# Future Self Simulator - Team Guidelines

## Project Overview

Future Self Simulator adalah pengalaman interaktif sinematik yang memungkinkan pengguna mengeksplorasi berbagai kemungkinan masa depan berdasarkan keputusan hidup yang mereka pilih melalui video AI-generated.

## Tech Stack

- **Framework**: Next.js 16+ (App Router)
- **Styling**: TailwindCSS 4
- **UI Components**: shadcn/ui
- **Animations**: Framer Motion
- **Video AI**: PixVerse
- **AI Workflow**: TRAE

## Tim Anggota & Tugas

### Anggota 1: Frontend & UI/UX
- Setup shadcn/ui
- Buat semua halaman UI
- Styling & animasi

### Anggota 2: Logic & Integrasi AI
- Decision Engine
- Integrasi PixVerse API
- State management

### Anggota 3: Video Production & Storytelling
- Storyboard
- Generate video dengan PixVerse
- Finalisasi video

## Git Workflow

1. Checkout ke branch `dev`
2. Buat feature branch dari `dev`
3. Commit dengan conventional commits
4. Buat Pull Request ke `dev`
5. Code review & merge
